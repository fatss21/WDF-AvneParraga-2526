Class selector

Doel: elementen selecteren op basis van een class attribuut.

Theorie

Een class gebruik je om meerdere elementen te selecteren die dezelfde stijl moeten krijgen. Een element met b.v. <... class="accent"> maak je als volgt op (let op het punt vooraf):

  padding: 6px;
}

Specificity

Doel: selectorgewicht begrijpen van tag-, class- en ID-selectoren.

Theorie

CSS kiest welke stijl wordt toegepast op basis van specificiteit, van hoog naar laag: ID > class > element. Voorbeeld voor <p class="blok id="belangrijk">: 

#belangrijk {
  color: blue; /* wint: #belangrijk is specifieker door dan .blok */
}
.blok {
  color: red; /* verliest: .blok is minder specifiek dan #belangrijk */
  font-size: 16px; /* wint: .blok is specifieker dan p */
}
p {
  font-size: 14px; /* verliest: p is minder specifiek dan .blok */
  margin-bottom: 14px; /* geen conflicten */
}

Attribute selector

Doel: selecteren op basis van attribuut(-waarde).

Theorie

Een attribute selector selecteert elementen op basis van een attribuut, en optioneel een waarde. Voorbeelden: 

input[placeholder] { /* alle input elementen met placeholder attribuut */
  border: 1px solid #aaa;
}
link[rel=stylesheet] { /* alle <link rel="stylesheet"> elementen */
  border: 1px solid #aaa;
}
[type=reset] { /* alle <... type="reset"> elementen */
  border: 1px solid #aaa;
}
img[src^="img/"] { /* alle img's waarvan de src begint met "img/..." */
  color: purple;
}

Pseudo class selectoren :hover, :active, :focus

Doel: elementen selecteren op basis van toestand.

Theorie

Een pseudo-class selector kan elementen selecteren op basis van toestand. De belangrijkste:

  • :hover – wanneer je met de muis over het element gaat
  • :active – wanneer je met de muis op het element klikt
  • :focus – wanneer je het element selecteert, b.v. door met de muis erin te klikken

Voorbeeld:

button:hover {
  opacity: 0.8;
}

Opdracht

  1. Geef de buttons bij hover een witte tekstkleur op zwarte achtergrond.
  2. Maak de tekst van de buttons bij active schuin met font-style:italic.
  3. Geef het tekstvak bij focus een lichtgele achtergrond met background-color: #ff9.
